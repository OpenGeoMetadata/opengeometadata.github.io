## Bounding Box

| Label                 | Bounding Box                                                                                                             |
|:--------------------- |:------------------------------------------------------------------------------------------------------------------------ |
| Aardvark-id           | 19                                                                                                                       |
| URI                   | `dcat_bbox`                                                                                                              |
| Obligation            | Recommended                                                                                                              |
| Multiplicity          | 0-1                                                                                                                      |
| Field type            | string                                                                                                                   |
| Purpose               | To support overlap ratio boosting in spatial searches                                                                    |
| Entry Guidelines      | This should be a bounding box in this format: ENVELOPE(w,e,n,s).                                                         |
| Commentary            |                                                                                                                          |
| Controlled Vocabulary | no                                                                                                                       |

| Element Set           | DCAT                                                                                                                     |
| Group                 | Spatial                                                                                                                  |
